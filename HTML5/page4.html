Listas de descripción HTML
HTML también admite listas de descripción.

La < dl>etiqueta define la lista de descripción, la < dt>etiqueta define el término (nombre) y la < dd> etiqueta describe cada término:

Coffee
- black hot drink
Milk
- white cold drink

Lista HTML desordenada: elija Marcador de elemento de lista

La list-style-typepropiedad CSS se utiliza para definir el estilo del marcador de elementos de la lista. Puede tener uno de los siguientes
valores:

Value Description
disc Sets the list item marker to a bullet (default)
circle Sets the list item marker to a circle
square Sets the list item marker to a square
none The list items will not be marked

< ul style="list-style-type:circle;">
